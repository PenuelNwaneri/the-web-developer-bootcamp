Club Bouncer

Are you old enough you get into the club?

Special conditions:

  • If age is negative, print an error message.
  • If age is 21, print "Happy 21st birthday! Bottoms up!"
  • If age is odd, print "Your age is odd."
  • If age is a perfect square, print "Your age is a perfect square!"
